1.2. Build tool – catkin (ubuntu 14.04 only)
Catkin is a powerful tool for building ROS components. For detailed documentation, please visit http://docs.ros.org/api/catkin/html/.
For Uos development, please download catkin_install.tar.gz and extract it; then use:
Please ask the administrator to mount NFS tool directory "/mnt/nas/public"

the way to mount:
sudo mkdir /mnt/nas/public -p
sudo apt-get install nfs-common 
sudo mount -t nfs -o nolock 192.168.100.7:/mnt/md0/public /mnt/nas/public

The way to make catkin make system work
cd /mnt/nas/public/catkin_install/
sudo apt-get install python-empy python-nose cmake libgtest-dev
sudo dpkg -i catkin_20160229-1_amd64.deb
sudo dpkg -i catkin-pkg_20160229-1_amd64.deb
sudo tar xvzf catkin-dev_20160229-1.tar.gz -C /opt

After that, please add below line to the end of your “~/.bashrc”:
source /opt/catkin/devel/setup.bash

1.3. Unit testing – gtest
Gtest is google’s C/C++ unit testsuits. It has been integrated with catkin and worked under catkin by default.
####下载源码解压后，用命令sudo make install直接安装

1.4. Version control – git
Please execute below commands to configure git:
git config --global user.name 
git config --global user.email 
git config --global core.editor vi
git config --global color.ui auto

Firstly, please refer to 2.2 to checkout a project source with git.
Then, modify a file, and use:
git commit 
To commit modified file to local git repository.
